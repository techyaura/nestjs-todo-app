## Description

TODO-APP created using [Nest](https://github.com/nestjs/nest) framework.

This app is using [POSTGRES](https://www.postgresql.org/)

## Installation

```bash
$ npm install
```

## Environment
```bash
create [.env.development] at root level for development mode if not using local Postgres connection with configuration as below:

PG_DB_HOST=<POSTGRES_HOSTNAME>
PG_DB_PORT=<POSTGRES_PORT>
PG_DB_USERNAME=<POSTGRES_USERNAME>
PG_DB_PASSWORD=<POSTGRES_PASSWORD>
PG_DB_NAME=<POSTGRES_DATABASE_NAME>
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
